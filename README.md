## 功能 / FEATURE 🎉

</div>

- ✅ 纯 `JavaScript`
- ✅ 集成 `Mock`、多页面等配置
- ⭕ 敬请期待……

<div align=center>

## 使用方法 / HOW-TO-USE 📖

</div>

```shell
# 先安装依赖 `npm i`（建议使用） OR `yarn install`

npm run serve
# 以 MOCK 模式启动（无需后台服务）
npm run serve:mock
# 打包（保存到 dist 目录）
npm run build
```
